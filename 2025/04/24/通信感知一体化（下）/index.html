1.旋翼无人机回波建模

1.1 单旋翼叶片单散射点回波模型

对于旋翼无人机，它的机械结构可简化成无人机主体和各个旋翼，每个旋翼由若干个旋翼叶片组成，旋翼叶片结构示意图如下图所示。

首先考虑旋翼上单个散射点的回波模型。当俯仰角为 0 时,考虑叶片上的一个散射点 P，该点的坐标位置为\((x_0,y_0)\)，散射点 P 到固定在机体的坐标原点，距离为 \[ l _ { p } = \sqrt { x _ { 0 } ^ { 2 } + y _ { 0 } ^ { 2 } } \qquad(1) \] 旋翼叶片转速为，则经过时间 t 后，散射点 P 的坐标位置为\((x_t,y_t)\)。初始旋转角为\(\varphi_0\)，时间 t 后， \[ \varphi _ { t } = \varphi _ { 0 } + \Omega t \qquad(2) \] 若旋翼沿雷达径向的速度为 v，时间 t 后，得到从雷达到该散射点的距离为 \[ R _ { p } ( t ) = \sqrt { R _ { 0 } ^ { 2 } + l _ { p } ^ { 2 } + ( v t ) ^ { 2 } + 2 l _ { p } R _ { 0 } \cos ( \varphi _ { 0 } + \Omega t ) + 2 R _ { 0 } v t + 2 v t l _ { p } \cos ( \varphi _ { 0 } + \Omega t ) }\qquad(3) \] 其中\(R_0\)是雷达到旋翼中心\(O\)的距离。假设在远场\(( \frac { l _ { p } } { R _ { 0 } } ) ^ { 2 } \rightarrow 0\)，上式可写为 \[ R _ { p } ( t ) \cong R _ { 0 } + v t + l _ { p } \cos ( \varphi _ { 0 } + \Omega t )\qquad(4) \] 雷达从散射点 P 接收到的信号为 \[ s _ { R } ( t ) = \text{exp} \left\{ - 1.2 单旋翼叶片整体模型

上节对旋翼的一个单散射点进行了回波建模，先对旋翼叶片整体进行建模。桨叶叶片的长度为\(L=L_2-L_1\)，\(L_2\)和\(L_1\)分别为旋翼中心到桨叶叶尖与桨叶根部的距离。则通过式(8)对叶片长度\(l_p\)积分，得到叶片整体总基带信号为 \[ \begin{aligned} s _ { L } ( t ) &= \sigma \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } \left[ R _ { 0 } + v t + \right] \right\} \int _ { L_1 } ^ { L_2 } \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } l _ { p }\cos\beta \cos ( \Omega t + \varphi _ { 0 }-\alpha ) \right\} d l _ { p }\\ & = \sigma (L_2-L_1) \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } \left[ R _ { 0 } + v t \right] \right\} \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } \frac { L_2+L_1 } { 2 } \cos \beta \cos ( \Omega t + \varphi _ { 0 }-\alpha ) \right\}\\ & \cdot \sin c \frac { 4 \pi } { \lambda } \frac { L_2-L_1} { 2 } \cos \beta \cos ( \Omega t + \varphi _ { 0 } -\alpha) \end{aligned}\qquad(9) \] 其中\(\sigma\)为叶片的 RCS，\(L\)是叶片的总长度。

对于有 N 个叶片的转子，N 个叶片有 N 个不同的初始转角 \[ \varphi _ { k } = \varphi _ { 0 } + k \frac { 2 \pi } { N } , k = 0 , 1 , . . . , N - 1\qquad(10) \] 结合式(9)可以得到单旋翼无人机的回波模型为： \[ \begin{aligned} s _ { N } ( t ) &= \sigma (L _ { 2 } - L _ { 1 } ) \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } [ R _ { 0 } + v t ] \right\} \times \\ &\sum _ { n = 0 } ^ { N - 1 } \sin c \left\{ \left[ \frac { 4 \pi }{\lambda}\frac{( L _ { 2 } - L _ { 1 } ) } { 2} \cos \beta \cos ( \Omega t + \varphi _ { 0 } + \frac { 2 \pi n } { N } - \alpha ) \right] \right\} \text{exp} \{ - j \Phi _ { n } ( t ) \} \end{aligned}\qquad(11) \]

1.3 多旋翼无人机回波模型

在此模型中假定无人机具有M个旋翼，每个旋翼具有N个叶片。一个旋翼的回波模型可以表示为每个叶片回波的线性叠加，多旋翼无人机回波模型可以表示为每个单旋翼回波的线性叠加： \[ \begin{aligned} s _ { \sum } ( t ) &= \sum _ { m = 1 } ^ { M } \sigma (L_2-L_1) \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } \left[ R _ { m } +vt \right] \right\} \times\\ &\sum _ { n = 0 } ^ { N - 1 } \sin c \left\{ \left[ \frac { 4 \pi } { \lambda } \frac { (L_2-L_1) } { 2 } \cos \beta _ { m } \cos ( \Omega _ { m } t + \varphi _ { 0 , m } + \frac { 2 \pi n } { N }-\alpha ) \right] \right\} \text{exp} \left\{ - j \Phi _ { m , n } ( t ) \right\} \end{aligned}\qquad(12) \] 式中，相位函数\(\Phi _ { m , n } ( t )\)为： \[ \Phi _ { m , n } ( t ) = \frac { 4 \pi } { \lambda } \frac { L_2+L_1 } { 2 } \cos \beta _ { m } \cos ( \Omega _ { m } t + \varphi _ { 0 , m } + \frac { 2 \pi n } { N } -\alpha), ( m = 1 , 2 , \cdots , M ; n = 0 , 1 , 2 \cdots , N - 1 )\qquad(13) \] 其中，\(R_{m}\)为某时刻第 m 个旋翼到雷达的距离，\(\beta_m\)为雷达与第 m 个旋翼之间的俯仰角，\(\Omega_m\)为第 m 个旋翼的旋转角频率，\(\varphi_{0,m}\)为第 m 个旋翼的某一叶片的初始旋转角。

一般地,对信号的相位函数求时间导数可以得到信号的瞬时频率。因此,对式(12)求时间导数可以得到第 m 个旋翼第 n 个叶片的等效瞬时多普勒频率\(f_{m,n}\)： \[ f _ { m , n } ( t ) = - \frac { 2 \Omega _ { m } } { \lambda }×\frac{L_2+L_1 }{2} \cos \beta _ { m } \sin ( \Omega _ { m } t + \varphi _ { 0 , m } + \frac { 2 \pi k } { N }-\alpha )\qquad(14) \] 对于一个悬停的旋翼无人机，旋翼到雷达的距离\(R_{m}\)与俯仰角\(\beta_m\)基本相等。

1.4 旋翼叶片的 RCS 模型

为了计算旋翼叶片的电磁散射，简单起见，叶片被简化为一个刚性的、均匀的、线性的矩形平板，它以恒定的转速绕固定轴旋转，并且不考虑前缘和后缘。在矩形平板上完全不考虑拍动、护板和加强筋。旋翼叶片和雷达的几何关系显示在上图中。在 RCS 公式中，有两个项：峰值 RCS \(\sigma_{Peak}\) 和方向因子\(\sigma_{Aspect}\)： \[ \sigma = \sigma _ { P e a k } \sigma _ { A s pect } = \frac { 4 \pi a ^ { 2 } b ^ { 2 } } { \lambda ^ { 2 } } ( \cos \theta \frac { \sin x _ { k } } { x _ { k } } \frac { \sin y _ { k } } { y _ { k } } ) ^ { 2 } \] 其中\(\sigma _ { Peak } = 4 \pi a ^ { 2 } b ^ { 2 } / \lambda ^ { 2 }\)，\(\sigma _ { A s p e c t } = ( \cos \theta ( \sin x _ { k } / x _ { k } ) ( \sin y _ { k } / y _ { k } ) ) ^ { 2 }\)，\(x_k=ka\sin\theta\sin\varphi\)，\(y_k=kb\sin\theta\cos\varphi\)，并且\(k=2\pi/\lambda\)。上式与极化无关，且仅对\(\theta\le 20°\)的小视角准确。

多旋翼可以由一组\(N_R\)个旋翼建模，其中每个旋翼都有\(N_B\)个叶片。为了计算\(N_R\)× \(N_B\)个旋转叶片的电磁散射，简单起见，可以将叶片建模为刚性的、均匀的、线性的矩形平板。对于完美导电的矩形平板，其 RCS 的数学公式如上式所示。为简单起见，每个叶片的 RCS 都可以分配到叶片的尖端。 \[ s _ { \sum } ( t ) = \sum _ { j = 1 } ^ { N_R } \sum _ { k = 0 } ^ { N_B-1 } \sigma _ { j , k } ( t ) \text{exp} \left\{ - j \frac { 4 \pi } { \lambda } R _ { j } ( t ) \right\} e x p \left\{ - j \Phi _ { j,k } ( t ) \right\} \] 其中\(N_R\)是是旋翼总数，\(N_B\)是每个旋翼上的叶片总数，\(R_j(t)\)是时间t从雷达到第j个旋翼中心的距离，\(\sigma_{j,k}(t)\)是时间t第j个旋翼上第k个叶片的RCS。

2.飞鸟目标的雷达回波建模

当雷达照射到飞鸟目标时，飞鸟翅膀的上下周期性振动也将在雷达回波信号中产生微多普勒效应，因此以雷达中心O为原点建立全局坐标系(X,Y,Z)，以飞鸟两侧翅膀的重合点o为原点建立飞鸟坐标系(x,y,z)，则雷达与飞鸟翅膀的相对位置如下图所示。设飞鸟中心在o点，其到雷达的距离为\(R_t\)，与雷达的方位角为\(\alpha\)，俯仰角为\(\beta\)，其相对雷达径向的飞行速度为v，飞鸟扑翼与xoy面的夹角为\(\phi_t\)。为方便分析，将飞鸟的扑翼看做是一个不弯曲的线源，它的回波信号可以由多个单散射点叠加而成，则可按照上一节节的做法对飞鸟进行分析。即先对飞鸟翅膀上的单一散射点进行分析，然后对单散射点的回波信号进行积分即可得到飞鸟目标的雷达回波模型。

取飞鸟翅膀上的一点P，设P点距飞鸟中心o的距离为\(l_p\)。在t时刻，雷达O到飞鸟中心o的距离为\(R_t=R_0+vt\)，飞鸟翅膀与xoy面的夹角\(\phi_t=\phi_{max}·sin\omega(t-\Delta t)\)，其中\(\phi_{max}\)是飞鸟翅膀与xoy面的最大夹角，\(\omega_b=2\pi f_{bird}\)是飞鸟翅膀震动的角频率，\(f_{bird}\)为鸟类的扑翼频率，\(\Delta t\)是用于表征飞鸟翅膀初始夹角的变量。
